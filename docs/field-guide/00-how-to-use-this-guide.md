# The Maintainer's Field Guide

This guide exists so that understanding of the codebase grows at the same rate as
the code itself. Each chapter lands alongside the work it explains, teaches the
concepts the code depends on, and ends with a guided reading list and
comprehension checkpoints.

How to use it:

- **Read the chapter first, then the referenced source files by hand.** The
  reading lists are short and ordered on purpose — they are the forcing function
  for real understanding, not the prose.
- **Checkpoints are self-tests.** If you can answer them without looking, you own
  that layer. If not, the answer is always in the listed files.
- Chapters are written against specific versions (AG Grid v36, Solid 2.0 betas)
  and updated when the ground shifts.
